### ☁️ [Cloud Portfolio Infrastructure](https://www.viveksubraveti.com)
Full-stack cloud-native portfolio with serverless backend — built entirely with IaC and CI/CD automation.

```
Route53 → CloudFront → S3 (Next.js)
                          ↓
                    API Gateway → Lambda (Python) → DynamoDB
```

**Stack:** Next.js 15 · TypeScript · Tailwind CSS · AWS (S3, CloudFront, Lambda, DynamoDB, Route53) · Terraform · GitHub Actions

**Highlights:** Interactive terminal, real-time visitor counter, dark/light theme, Framer Motion animations, modular Terraform infrastructure

### 🚗 Vehicle Maneuver Analysis Platform
Geospatial analysis platform for autonomous driving research — detecting acceleration, turns, and junction events from route data.

**Stack:** Python · OSMnx · NetworkX · Plotly · Folium · OpenStreetMap

**Highlights:** 95% classification accuracy · 1,000+ km of route data analyzed · 50% processing time reduction

---

### 🧠 ADAS Data Optimization (Master Thesis)
Hybrid computer vision and deep learning pipeline to reduce raw ADAS camera data volume by filtering non-traffic-relevant regions.

**Stack:** Python · OpenCV · TensorFlow · PyTorch · NumPy

**Highlights:** Semantic segmentation for autonomous driving datasets · Combined classical CV with deep learning
